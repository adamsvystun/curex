### Third way

Direct instalation
Requirments:
* Django 1.11
* Python >3.5

```
git clone https://github.com/adamsvystun/curex.git
pip3 install plotly
cd curex
python3 manage.py migrate
python3 manage.py createcachetable
python3 manage.py runserver
```

(Checked on Linux Deepin 4.9.0-amd64)
